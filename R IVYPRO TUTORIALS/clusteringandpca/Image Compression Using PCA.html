Image Compression Using PCA¶

Dimensionality Reduction is a form of unsupervised learning that helps us reduce the size of data that would have been computationally expensive if used in it's raw form. One of the applications of dimensionality reduction involves image compression. The compressed image can then be used for further analysis. Principal Component Analysis is one of the main driving factors behind implementing Dimensionality Reduction

Standardize the Data¶

This is done to bring the features to scale

X_std = (img_grayscale-mean(img_grayscale.T,axis=1))

Apply Principal Component Analysis¶

In [8]:
Y_sklearn = PCA(n_components=20,whiten=True).fit(X_std)
Y_sklearn = Y_sklearn.components_.T

Recover Data from Eigen vectors¶

In [9]:
score = dot(Y_sklearn.T,X_std.T) # project 
Ar = dot(Y_sklearn,score).T + mean(img_grayscale,axis=0) # recover
